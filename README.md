This repository contains my capstone project on **Autonomous Driving**, focused on applying **Machine Learning and Deep Learning** techniques to solve real-world self-driving car problems.  
The project demonstrates an end-to-end AI pipeline for perception and driving-assistance tasks.

---

## Project Overview
The goal of this project is to build intelligent systems that can understand road scenes and assist autonomous driving decisions.

Key components include:
- Lane Detection using Computer Vision
- Steering Angle Prediction using Deep Learning
- Traffic Sign Recognition
- Behavioral Cloning for end-to-end driving
- Visual perception with Convolutional Neural Networks (CNNs)

---

## Tech Stack
- **Programming Language:** Python  
- **Libraries:** NumPy, Pandas, OpenCV  
- **Deep Learning:** TensorFlow, Keras  
- **Models:** Convolutional Neural Networks (CNNs)

---

## Project Pipeline
1. Data collection and preprocessing  
2. Image normalization and feature extraction  
3. Lane detection using computer vision techniques  
4. Steering angle prediction using CNN models  
5. Traffic sign classification  
6. Behavioral cloning for driving behavior learning  
7. Model evaluation and performance analysis  

---

## Problem Statement
Design and implement an **end-to-end AI system** capable of interpreting road scenes and supporting autonomous driving decisions under real-world conditions.
